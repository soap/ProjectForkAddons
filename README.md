ProjectForkAddons
=================

This is a ProjectFork (Joomla! extensions) addons package comprises two plugins using to force automatic setting of default project based on data on user profile.

Requirements
------------

  Joomla! 2.5.x
  
  ProjectFork 4.x
  
  PHP 5.2.4+

Testing on Joomla 3.x is welcome.
